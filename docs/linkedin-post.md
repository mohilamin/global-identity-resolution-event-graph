# LinkedIn Post Draft

I built a new portfolio project: Global Identity Resolution & Attribution Event Graph.

It is not a generic ETL pipeline. It simulates a FAANG-style identity resolution problem: stitching fragmented synthetic events across cookies, devices, sessions, anonymous users, known users, payments, support accounts, ad clicks, purchases, and subscriptions.

What it includes:

- deterministic and probabilistic identity matching
- identity graph nodes, edges, and clusters
- session stitching
- multi-touch attribution paths
- suspicious identity cluster detection
- link explanations and lineage
- DuckDB warehouse
- FastAPI endpoints
- Streamlit dashboard
- pytest coverage
